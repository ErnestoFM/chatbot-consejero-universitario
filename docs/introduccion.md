# Introducción

El **Chatbot Consejero Universitario** es una aplicación web que utiliza IA para orientar a estudiantes universitarios en trámites, asesoría emocional, quejas y orientación vocacional. Integra la API de Gemini de Google y permite adjuntar archivos para enriquecer las consultas.

- Interfaz tipo ChatGPT
- Temas iniciales predefinidos
- Soporte para archivos (PDF, imágenes, videos)
- Historial de conversaciones
- IA con Gemini
- Preparado para RAG (Retrieval-Augmented Generation)
